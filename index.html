O que é display flex?

É uma propriedade CSS, sendo um contêiner flexível, usado para organizar elementos filhos.

O que é justify-content?

É uma propriedade CSS, usada para alinhar itens ao longo do eixo x (horizontal) de um contêiner.

O que é align-items?

É uma propriedade CSS, usada para alinhar itens ao longo do eixo y (vertical) de um contêiner.
